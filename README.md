# ShoePlan

A personalized shoe recommendation and care service based on your running style and needs.

### MVP (Minimum Viable Product)

본 서비스의 MVP는 사용자가 자신의 러닝 목적과 스타일을 입력하면,
이에 적합한 러닝화를 추천 받을 수 있으며, 자신의 현재 러닝화의
교체 시기를 확인할 수 있도록 하는 것을 목표로 한다.

**MVP 단계에서 제공하는 핵심 기능**

- 사용자가 러닝 목적(거리, 빈도, 장소 등)을 입력 할 수 있다.
- 현재 러닝화의 쿠션 수명을 기반으로 교체 시기를 안내한다.
- 입력된 정보를 기반으로 사용자에게 적합한 러닝화를 추천한다.
- 각 러닝화의 기본 정보(브랜드, 모델, 특징)를 확인할 수 있다.
